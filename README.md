# Smart-Campus-Management-System
这款智慧校园管理微信小程序，是校园生活的得力助手。学生能一键查询课表、成绩，快速完成请假、报修，还可参与校园活动报名，查看图书馆借阅信息。教师借助它发布作业、批改试卷、管理班级事务、接收学校通知。家长能实时了解孩子在校动态，包括考勤、学业情况等。小程序还支持校园缴费、校车预约，集多样功能于一体，为师生及家长打造便捷、高效的校园体验 。
